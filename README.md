# OpenAI python app

OpenAI chatGPT app.



## Requirements

 - OpenAI



## Setup

 - After cloning: cd myai
 - python3 -m venv myvenv
 - . myvenv/bin/activate
 - pip3 install -r requirements.txt



## Using

 - Get API-KEY from openai.com
 - Set environment variable: export OPENAI_API_KEY=<your_key_here>
 - Just run:  main.py
